1. Tables

Tables are used to display data in rows and columns. They are structured using the following tags:

  • <table> → Defines the table
  • <tr> → Defines a table row
  • <th> → Defines a table header (bold, centered by default)
  • <td> → Defines a table cell (data)
  • <caption> → Adds a title to the table

2. Forms

Forms allow users to enter and submit data. They are built using these common tags:

  • <form> → Defines the form area
  • <label> → Describes input fields
  • <input> → Creates input fields (text, email, password, etc.)
  • <textarea> → Creates multi-line text area
  • <select> <option> → Creates a dropdown menu
  • <button> or <input type="submit"> → Submits the form

1. Text & Headings

Used to format text and create headings on a webpage.

  • <h1>...</h1> → Main heading (largest)
  • <p>...</p> → Paragraph
  • <strong>...</strong> → Important/bold text
  • <em>...</em> → Emphasized/italic text
  • <br> → Line break

2. Lists

Lists are used to group related items.

  • <ul> → Unordered list (bullets)
  • <ol> → Ordered list (numbers)
  • <li> → List item
  • <dl>, <dt>, <dd> → Description list

3. Links & Media

Used to add hyperlinks, images, and multimedia content.

  • <a href="..."> → Hyperlink
  • <img src="..." alt="..."> → Image
  • <audio> & <video> → Multimedia files
  • <figure> & <figcaption> → Image with caption

4. Tables

Tables organize data into rows and columns.

  • <table> → Table wrapper
  • <tr> → Table row
  • <th> → Table header
  • <td> → Table cell
  • <caption> → Table title

5. Forms

Forms allow user input.

  • <form> → Form wrapper
  • <input> → Input field (text, email, etc.)
  • <textarea> → Multi-line input
  • <select> & <option> → Dropdown menu
  • <button> → Clickable button

6. Semantic Tags

These give meaning to content and improve SEO.

  • <header> → Page or section header
  • <nav> → Navigation links
  • <main> → Main page content
  • <section> → Content section
  • <article> → Independent article
  • <aside> → Sidebar/extra info
  • <footer> → Page footer
